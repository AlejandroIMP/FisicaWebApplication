# Análisis del Problema

Este ejercicio involucra el **encuentro de dos móviles** que se mueven en direcciones opuestas con velocidades constantes.

## Estrategia de Solución

Para resolver este tipo de problemas:
1. Establecer un sistema de referencia
2. Escribir las ecuaciones de posición para cada móvil
3. Igualar las posiciones en el momento del encuentro
4. Resolver para encontrar el tiempo y la posición

## Conceptos Clave

- **Velocidad relativa**: Cuando dos objetos se mueven en direcciones opuestas, sus velocidades se suman
- **Punto de encuentro**: Posición donde ambos móviles coinciden en el mismo instante
