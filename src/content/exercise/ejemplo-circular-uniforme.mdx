# Movimiento Circular Uniforme

El **movimiento circular uniforme (MCU)** es el movimiento de un objeto que describe una trayectoria circular con velocidad angular constante.

## Características del MCU

- **Trayectoria**: Circular
- **Velocidad angular**: Constante (ω = constante)
- **Velocidad lineal**: Magnitud constante, dirección variable
- **Aceleración**: Solo centrípeta (dirigida hacia el centro)

## Magnitudes Fundamentales

### Velocidad Angular (ω)
- **Definición**: Rapidez de cambio del ángulo
- **Unidades**: rad/s
- **Relación**: ω = θ/t = 2π/T = 2πf

### Período (T) y Frecuencia (f)
- **Período**: Tiempo para una vuelta completa
- **Frecuencia**: Número de vueltas por unidad de tiempo
- **Relación**: f = 1/T

### Velocidad Lineal (v)
- **Definición**: Velocidad tangencial del punto
- **Relación**: v = ωr
- **Dirección**: Tangente a la circunferencia

### Aceleración Centrípeta (aᶜ)
- **Definición**: Aceleración hacia el centro
- **Fórmulas**: aᶜ = v²/r = ω²r
- **Dirección**: Hacia el centro del círculo

## Aplicaciones

- **Ruedas y engranajes**
- **Movimiento de satélites**
- **Centrifugadoras**
- **Parques de diversiones**
- **Átomos y moléculas**
