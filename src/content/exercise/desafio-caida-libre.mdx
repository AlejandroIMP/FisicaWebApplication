## Instrucciones para Resolver

Este es un problema de **caída libre**, un caso especial de movimiento uniformemente acelerado donde:

- La aceleración es constante e igual a g (9.8 m/s²)
- La velocidad inicial puede ser cero (objeto dejado caer) o diferente de cero (objeto lanzado)
- No se considera la resistencia del aire

### Pasos Sugeridos

1. **Identifica los datos conocidos y las incógnitas**
2. **Selecciona las ecuaciones apropiadas**
3. **Sustituye los valores conocidos**
4. **Resuelve para cada incógnita**
5. **Verifica tus resultados**

### Ecuaciones de Caída Libre

Para un objeto en caída libre, las ecuaciones cinemáticas se simplifican:

```
h = v₀t + ½gt²
v = v₀ + gt  
v² = v₀² + 2gh
```

Donde:
- **h** = altura o distancia recorrida
- **v₀** = velocidad inicial
- **v** = velocidad final
- **g** = aceleración gravitacional (9.8 m/s²)
- **t** = tiempo

### Estrategia de Resolución

> **Sugerencia**: Comienza por la ecuación que relaciona directamente los datos que tienes con la incógnita que buscas.

¿Qué ecuación usarías primero para encontrar la altura? ¿Y para la velocidad final?

### Reflexión

Una vez que resuelvas el problema, piensa en:
- ¿Qué pasaría si el edificio fuera más alto?
- ¿Cómo cambiarían los resultados en la Luna (g = 1.6 m/s²)?
- ¿Es realista despreciar la resistencia del aire para esta altura?
