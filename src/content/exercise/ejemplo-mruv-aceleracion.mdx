# Movimiento Rectilíneo Uniformemente Acelerado (MRUV)

El **MRUV** es un tipo de movimiento donde la aceleración permanece constante, lo que significa que la velocidad cambia uniformemente con el tiempo.

## Características del MRUV

- **Aceleración constante**: a = constante ≠ 0
- **Velocidad variable**: La velocidad cambia linealmente con el tiempo
- **Trayectoria rectilínea**: El movimiento ocurre en línea recta

## Ecuaciones Fundamentales

Las tres ecuaciones cinemáticas del MRUV son:
1. **v = v₀ + at** (relaciona velocidad, tiempo y aceleración)
2. **x = v₀t + ½at²** (relaciona posición, tiempo y aceleración)
3. **v² = v₀² + 2ax** (relaciona velocidad, posición y aceleración)

## Aplicaciones Comunes

- Automóviles acelerando o frenando
- Objetos en caída libre
- Proyectiles (en cada componente por separado)
